## API Deployment Instructions ##
Use the below command to push to heroku prod

`git push heroku main`

### Set these ENV variables ###
```
PORT
SPOTIFY_CLIENT_ID
SPOTIFY_CLIENT_SECRET
API_DOMAIN
WEB_APP_DOMAIN
```
